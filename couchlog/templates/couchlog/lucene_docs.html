Advanced search lets you do freetext searches as well as searches on specific fields using the lucene query syntax. Review the documentation there for detailed information. For couchlog, the message, type, and url are searched by default. Additionally the following fields can be searched:

Field name Description
date date and time of error.
level log level of error.

Examples

Here are some search examples (paste these into the search bar to see results).

Search all documents that are errors or warnings:

level:ERROR OR level:WARNING

Search all documents matching "files" in october:

files AND date:[2010-10-01 TO 2010-11-01]
