Advanced search lets you do freetext searches as well as searches on specific fields using the lucene query syntax. Review the documentation there for detailed information. For couchlog, the message, type, url, and extra info are searched by default. Additionally the following fields can be searched:

Field name Description
clinic Clinic code (e.g. "5010140")
version App version (e.g. "0.1.0")
commit Commit id (hash)
date date and time of error.

Examples

Here are some search examples (paste these into the search bar to see results).

Search all documents from two different clinics:

clinic:5010140 OR clinic:5010120

Search all documents matching "files" in october:

files AND date:[2010-10-01 TO 2010-11-01]

Search all documents matching a certain version, but not a certain commit:

version:0.1.0 AND NOT commit:fe272d2cd55a0dfc6b4df33c73e39a79d2a515d8
